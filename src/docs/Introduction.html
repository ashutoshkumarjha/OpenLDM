Introduction

Open-source Land-use Land-cover Dynamics Modeling Platform (OpenLDM) v1.0 is a comprehensive Land Use/Land Cover (LULC) Modeling environment where the modeler has been provided to apply different modeling approach (Logistic, Linear, Neural Network and Random Forest based) along with the accuracy assessment and data visualization under single platform. It is built on the principle of "Select-Carry-Assess-See".


This software is developed as an extension of requirements felt for land change analysis and modeling for Indian river basins, at Indian Institute of Remote Sensing ,Dehradun (IIRS) under the ISRO Geosphere Biosphere Program(ISRO-IGBP) project.This platform is open source extended version of ILULC-DMP model developed under ISRO-Geosphere Biosphere Programme by Indian Institute of Remote Sensing (IIRS), Indian Space Research Organisation Dehradun, INDIA.
