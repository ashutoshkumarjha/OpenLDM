Data Selection (T0)

This section allows the user to find out the dependencies of drivers and its class under different modeling techniques. Based on this user can select one of the models. It shows the model parameters and its details statistical analysis and results. It has following two subsections.

      1. Select Driver

       Under this section user is given to add the number of drivers based on the availability of drivers. The raster file corresponding to the drivers can be selected by pressing first pressing “Add Drivers” and then press the “Select Drivers” button in “Select Drivers” table. The model assumes that there are minimum 3 drivers to be used for the modeling purpose. After clicking Add Drivers button three times the bottom section of “Do Model Fitting” will be enabled.

Users can provide different names to each of drivers under the Driver Name columns. The removal of any drivers can be done by first selecting the last drivers for the row which needs to be changed and then deleting the last row.


      1. Do Model Fitting

This section allows the evaluation of different modeling techniques. Selection of radio button against each modeling technique will enable the “View Model Statistics”. The three modeling techniques are available as of now. The logistic regression models are considered to be good for the Land Use modeling. Based on the selection of models the statistics can be seen in the “Model Parameter Output” box. The “Model Parameter Output” box provides the coefficient of each of the drivers, its significance and residual error distribution along with the model fitting statistics.
