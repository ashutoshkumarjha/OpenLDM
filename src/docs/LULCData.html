Inside the path example\LULC\ you willfind following files

    1. example\LULC\1985.tif  (LULC classified file for year 1985)
    2. example\LULC\1995.tif  (LULC classified file for year 1995)
    3. example\LULC\2005.tif  (LULC classified file for year 2005)


The LULC data are prepared after vector to raster conversion of LULC map available at scale of 1:250,000. The original LULC vector maps were generated based on visual interpretation of satellite images. Classified map attributes are is listed in Table 1
