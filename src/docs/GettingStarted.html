Getting Started

This software has polymorphic distribution and supported on Windows, Linux and Macintosh.

It has a plugin in QGIS and also works in command line apart from a standalone application. The software has dependencies on R and R library such as

plotting and image generation library - png, grid

Geospatial Library - raster, maptools, rgeos, rgdal .

Parellal Processing Library - parallel, doParallel

Modeling libary -nnet, randomForest

Data Organisation Libray - data.table

In case of GUI based distribution such as QGIS plug-in or standalone distribution it has additional dependencies of Python,rpy2 and PyQt. This software has been tested with following version of software

PyQt  4.8.5

QGIS 1.8.0

Python 2.7 / 3.4

R 2.15 / 3.0.1

Rpy2
