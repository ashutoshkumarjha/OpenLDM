Getting Started

This software has polymorphic distribution and supported on Windows, Linux and Macintosh.

It has a Plugin in QGIS. It works on command line. There is also a standalone application built using the same framework.In case of GUI based distribution such as QGIS plug-in or standalone distribution it has additional dependencies of Python,Rpy2 and PyQt. This software has been tested with following version of software. The software has major dependencies on R and R library such as

    1. R 2.15 / 3.0.1
    2. Plotting and image generation libraries - png, grid

    3. Geo-spatial libraries - raster, maptools, rgeos, rgdal .
    4. Parallel Processing libraries - parallel, doParallel
    5. Modeling libraries - nnet, randomForest
    6. Data Organization library - data.table
