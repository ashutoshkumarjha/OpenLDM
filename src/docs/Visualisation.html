The LULC modeling platform also has provision of basic map visulisation.It provides the ability to generate a map with title, legend and color scheme along with essential map elements like north arrow and scale.

To view a map you need to add the file into file combo box.Click for on the “Show” button. In response a message dialogue box “File Doesn’t exist” will pop up with message “Do you want to add a file?”.Click on “Yes” button as shown in Figure 21.This will open the “Select File” dialogue. Browse to the location of predicted file. After file selection the map will be plotted in plotting area as shown in Figure 22.Please see the colour scheme of the map is gray scale.

We can change the colour scheme by clicking on colour field  against each class. Click on one of the colour field of Builtup class and choose red color in “Color Picker”. This step is shown Figure 23. Repeat this step for each LULC classes.

After setting colours for each LULC class click on the “Show” button. This will result in new color scheme to be applied to entire map. The result is shown in Figure 24.
