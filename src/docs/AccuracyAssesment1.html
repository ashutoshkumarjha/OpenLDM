Accuracy Assesment

Modeling Example ›› Execution ››

This section has two subsections. It has two mandatory inputs. These are predicted File and Reference File. The functionality of this tab can be used independently also. The detail descriptions of different sub sections are given below.

    1. Summary

After providing the reference and predicted files and clicking on “Find Accuracy” button  different fields in this sub section get populated. The different fields are self explanatory. These fields proved the computed overall accuracy and overall kappa accuracy. The ranges of these metrics are also with 95% and 99% confidence interval in their respective fields.

    1. Detailed

This subsection has provision of getting the confusion matrix and class wise user’s and producer’s accuracy. These metrics get populated into table the based on selection of radio button.

After clicking the “Execute” button at the bottom of “Output Generation” screenthe generated output file will added to the “Predicted File” section.Please add the reference file by clicking“Select File” button provided in front of “Reference File”. In the “Open File” dialogue box browse to the folder examples/LULC and select the File 2005.tif

After the selection of reference file click on the “Find Accuracy” button in “summary” section tab.The accuracy result will get populated. As a result the screen will look alike as Figure 18.
