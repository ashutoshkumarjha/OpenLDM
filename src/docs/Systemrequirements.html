System requirements

Getting Started ››

This software can be run on any platform which is capable of running the R and Python. The system hardware requirement is completely dependent upon data volume. The software has dependencies on R for command based version. For GUI version it has dependencies on Python and/or QGIS depending upon you use GUI/QGIS Plug-in.

For Anaconda Environment

Step 1. Install Anaconda as per the instruction given.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Please install all the dependencies as given below depending upon the OS.

For Windows*

Step 1. Installing R and its libraries.

1.1.1  Go to R official site and download the latest version of R. e.g. R-3.0.2-win.

Note:  It is always better to choose slightly earlier version to avoid lack of libraries for the current version. While R may be latest but the library may be under development. The older version can be found at http://cran.r-project.org/bin/windows/base/old. Click on R- 3.0.1 to download the file.

     1.1.2 After installing the R, invoke the R and install the libraries ('png', 'grid' ,'raster' ,'maptools' ,'rgeos' ,'rgdal' ,'parallel' ,'doParallel' ,'nnet' ,'data.table') by copying and pasting following command on R prompt.

       > install.packages(‘packageName’)


Step 2. Click on the installer and follow the instruction to install the Program. Next step is required only incase where you want to perform the development.


Step 3.To be followed for GUI. Please install python and associated libraries as given below.Click the link to download the file directly.

2.1 Python 2.7 for python2.7-32bit or python2.7-64bit depending upon your OS.

2.2 rpy2 for python 2.7 for rpy2-32bit or rpy2-64bit depending upon your python version.Rpy2 version installer can be found here.

2.3 pywin32 for python 2.7 for pywin32-32bit or pywin32-64bit depending upon your python version.

2.4 PyQt4 for python 2.7 for PyQt4-32bit or PyQt4-64bit depending upon your python version.

*Required only for development version.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

For Ubuntu Linux

Step 1. Open terminal.

Step 2. Installing R and its libraries use the following command.

       2.1.1 #sudo apt-get install r-base  

               For setting the repository follow the instruction here.

       2.1.2 After installing the R invoke the R and install the following a few libraies by copying and pasting following command on R prompt.

       > install.packages('png','grid','raster','maptools','rgeos','rgdal','parallel','doParallel','nnet','data.table')


Step 3.To be followed for GUI. Please install python and associated libraries as given below.

       2.2.1 #sudo apt-get install python-rpy2
