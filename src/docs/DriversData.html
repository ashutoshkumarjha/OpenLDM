Drivers data for year 1985/1995/2005

Inside the path example\Drivers\  we see following four folders

    1. examples\Drivers\Drivers85\  (LULC classified file for year 1985)
    2. examples\Drivers\Drivers95\  (LULC classified file for year 1995)
    3. examples\Drivers\Drivers05\  (LULC classified file for year 2005)
    4. examples\Drivers\DriversCommon  (Drivers which are constant)

Each of these folder contains following drivers.

    1. distanceToBuiltUpYY.img
    2. distanceToForestYY.img
    3. distanceToWaterbodyYY.img
    4. populationDensityYY.img
    5. rainYY.img
    6. temperatureYY.img
    7. distanceToRoads.img
    8. elevation.img
    9. slope.img
    10. soil.img

Where YY represents the year such as (85,95,05).
