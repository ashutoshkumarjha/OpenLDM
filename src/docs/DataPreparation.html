There are five mandatory inputs which need to be provided to go the next stage. These are marked in red The different parts of this TAB is explained below. The interface is divided into following section.

  1. Input Section:

It has two subsections.

    1. Project Section

The project section sets the parameters to entire project. It has project directory which acts as the initial directory to search whenever any data inputting is performed. It also sets the NA value to be considered in the input file.

    1. Data Input Section

This section has four mandatory fields and optional fields. The mandatory fields as marked in Figure 3. The optional fields are “Area Of Interest” and “Mask”.The “T1 File” field requires the initial model input file. The “T2 File” field requires second year input. The “Year (T1)” and “Year (T2)” fields correspond to the year for which the two files have been supplied.

  1. Output Section

This section has one mandatory field. “Output File Year (T3) ” field corresponds to the name of output predicted file which needs to be generated.
