This tab provides the summary of the coefficients and its significance as estimated in the previous steps. Here we can add or remove the model coefficients as per user analysis. The selected drivers will be used for modeling purpose based on Akaike’s An Information Criteria (AIC) scores of model. The has two subsections


    1. View Model Coefficient

       This section allows selection of individual drivers for a specific class .User can do the selection of each driver based on observation of the significance level of each driver related to a particular class. These coefficients are used for the generation of suitability maps for each class.

       We can also label DN numbers in the input LULC file as per the user's choice.


    1. Migration Order

       Under this section user can set the priorities of classes to be considered in case of two or more classes contribute to a particular class during the duration between year T1 and T2.It has two option “Auto” and “User Defined”. Under “Auto” selection the transition probability is used to set the order. The “User Defined” mode requires the order to be given in positive integer values starting from 1,2,…n. The smaller the value the earlier the selection of that class will be for the allocation to meet the demand
